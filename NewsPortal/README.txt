Учебный Django проект - новостной портал

Структура
1) news_portal:
	board_news - приложение, реализующее вывод списка новостей и отдельной новости.
	fpages - приложение, расширяющее возможности админ-панели. 
	news_portal - директория с настройками всего проекта.
	static - директория с шаблонами css.
	templates - директория с HTML файлами, устанавливающими формат отображения информации пользователю.
	db.sqlite3 - база данных проекта.
2) venv - виртуальное окружение проекта, содержащее все необходимые библиотеки.

Все новости выводятся в порядке убывания даты добавления по адресу /news в формате:
заголовок, дата и время добавления, первые 20 символов текста.

Отдельная новость выводится по адресу /news/{id} в формате:
заголовок, дата добавления, текст статьи.

Заголовок и текст проходят цензуру с помощью кастомного django-фильтра censor
(board_news/templatetags/custom_filters.py)
Слова, подлежащие цензуре, лежат в множестве 'SET_BAD_WORDS'.
Определение 'плохих' слов происходит по нормальной форме слова (библиотека pymorphy2)
